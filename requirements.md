# Project Goal

A complete property selling/renting platform. Where user can create their account and buy/rent property. Agent can create their account and take a subscription plan and list property to sell/rent.

# Role (USER)

1. User Authentication
2. Buy/Rent property - with filtering
3. Add to favorite/wishlist
4. Simple chat with agent and negotiation with reference of property

# Role (AGENT)

1. User Authentication
2. Subscription purchess for Agent property listing
3. Property listing
4. Simple chat with user and negotiation with reference of property

# Role (SYSTEM_ADMIN)

1. Only login
2. User/Agent management
3. Property management
4. Subscription management
5. View all contact with chats

# Website (Public)

1. Home page
2. Property listing page
3. About us page
4. Contact Us page
